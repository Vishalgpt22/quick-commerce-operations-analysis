# Quick Commerce Operations Performance Analysis

This project analyzes quick-commerce delivery operations to understand
the impact of delivery speed, distance, discounts, and city performance
on customer ratings and order value.

## Tools Used
- Excel (Data Cleaning & Feature Engineering)
- SQL (Aggregation & Validation)
- Python (Pandas, NumPy â€“ Correlation Analysis)
- Tableau (Interactive Dashboard)

## Key Insights
- 97% of orders are delivered within 30 minutes
- Delivery time increases moderately with distance
- Customer ratings vary more by city than delivery speed
- Discounts increase order value but not customer ratings
